Rendering movies

  1. Composition > Add To Render Queue

  2. Render Settings > Best Settings
    • Quality: Best
    • Resolution: Full
    • (Defaults may be fine otherwise; take a look)
  3. Output Module > Lossless
    • Format: H.264 (for Vimeo)
    • Check Audio Output box

  4. Output To Browse to location for Output

  5. Click on Render (upper right hand corner of Queue panel)

Preparing for Vimeo

  • All assignments must be posted onto your Vimeo account. This is where we will view them. Please note:
    • Sometimes it can take several hours for a video to upload so plan accordingly
    • Make sure you properly compress videos for upload to keep a smaller file size. Generally speaking using the CODEC h.264 is recommended (NOT animation).
  • More about the Codec: H.264:
    • A codec is the format in which your video will be encoded. Vimeo accepts most major video codecs, but for best results we recommend using H.264. If you’re uploading High Definition (HD) video, choose the High Profile H.264 setting instead of Main Profile.
    • Advanced H.264 settings: To make your uploads extra smooth, choose the following advanced settings whenever possible.
      • Closed GOP
      • CABAC (to reduce your file size)
  • Vimeo’s compression guide

AE Keyboard Shortcuts

  • To view/edit properties of a selected layer
    • A: anchor point
    • P: position
    • S: scale
    • R: rotation
    • T: opacity
    • U: view only layers with keyframes
    • E: view only effects
    • Shift: Holding down shift key allows to see multiple properties.
      e.g. Pushing P, Shift + S, Shift + R will bring up position, scale, and rotation
    • Option: Holding down option key will add a first keyframe while bringing it up.
      e.g. Option + P will open the position property and add a keyframe at the same time. This is only with the first keyframe.
  • To edit layer on the timeline
    • [: Move the IN point of the layer to the current timeline marker
    • ]: Move the OUT point of the layer to the current timeline marker
    • Option+[: Cut the IN point of the layer to the current timeline marker
    • Option+]: Cut the OUT point of the layer to the current timeline marker
  • To adjust workarea
    • B: Set the beginning of workarea to the current timeline marker
    • N: Set the end of workarea to the current timeline marker
    • Remember IBON (In, Beginning, Out, eNd)
  • AE Keyboard Shortcuts reference

Shortcuts for non-numerical keyboards (i.e. laptops)

  • RAM preview with audio: Control + 0 (zero)
  • Mark audio: Control + 8
  • Manually preview audio: Drag current timeline marker while holding down Command key
  • Preview only audio from current timeline: Control + . (period)
  • Preview only audio from current work area: Control + Option + . (period)

Tech Demo 3

Outlines and instructions for in-class tech demon

Download these files to use during in-class tech demo.

Note: This demo shows you how to make GIFs using Photoshop. There are other ways to make GIFs—lots of people who don’t have access to Photoshop use online video to GIF conversion sites, etc—but Photoshop is definitely the most powerful GIF-making software at your disposal.

Making a GIF out of multiple image files

  • Import the series of photos as individual layers:
    • Go to File > Scripts > Load Files into Stack
    • Select all of the images you want to use. Make sure they’re arranged in the correct order.
    • Click OK, and each photo will be imported as its own layer.
  • Once you have imported each image as a layer:
    • Open the Timeline window
    • Click the Create Frame Animation button in the Timeline window
    • Click the Timeline options dropdown button in the window’s top right corner and select Make Frames From Layers
  • You should now have one frame for each layer in your Photoshop file.
    • Using the controls on the bottom of the window you can preview your GIF, delete frames, and decide whether your animated GIF plays once, loops forever, or plays a specific number of times.
    • Change the number beneath each frame (“___ seconds”) to determine how much of a time delay there is from frame to frame.
    • Reorder the frames by clicking and dragging
  • To export your gif from Photoshop CS6 or earlier:
    • Go to Photoshop’s top menu bar and select File > Save for Web
    • In the Save for Web Popup window, make sure the selected file type is Gif
    • You can adjust the number of colors, image size, and more in this window
    • Click Save
  • To export your gif from Photoshop CC:
    • Go to Photoshop’s top menu bar and select File > Export > Save for Web (Legacy)
    • In the Save for Web Popup window, make sure the selected file type is Gif
    • You can adjust the number of colors, image size, and more in this window
    • Click Save

Making a GIF from an Illustrator project

Illustrator does not have the animation capabilities of Photoshop. So you can work in Illustrator, but you’ll have to export all of your individual frames as images and place them in Photoshop as individual layers.

  • Create each frame of your animated gif as a separate Artboard in the same Illustrator file. Each Artboard should be the same size.
  • Once your frames are ready, go to Illustrator’s top menu bar and select File > Export.
  • In the Export popup window, make sure the box next to Use Artboards is checked, and make sure you’re exporting all of your Artboards.
  • Choose a file format—the specific type is not important: png, jpg, etc. all are fine—and hit Export.
  • Once finished exporting all of your frames as images, finish making your gif using the same method outlined above

Modifying a pre-existing GIF

  • Drag a preexisting animated GIF into Photoshop to open it up. Each frame will import as a separate layer.
  • Ignore the frames in the Animation window for now.
  • Make whatever additions/alterations to EVERY individual layer. Remember that each layer represents one frame of your animated gif, so if you want to add an element to every frame it must be added to every layer. If you have modified things by adding new layers on top of existing layers, you will need to combine these new layers with the existing layers before proceeding.
  • When your layers are the way you want them to be, delete the existing frames in your animation panel. Altering the layers has likely messed up the existing animation.
  • Use the Timeline options dropdown to Make Frames from Layers and proceed to finalize your animated GIF as usual—setting each layer’s time delay, determining the number of loops, etc—and Save for Web.

Making a GIF from video footage or animation

  • Open Photoshop, go to the top menu bar and select Import > Video Frames to Layers.
  • Select your video file, and use the Import Video to Layers popup window to decide how much of the footage to import.
    • You can choose to only import some of the frames (e.g. every 2 frames, every 3 frames). Using fewer frames will make the file smaller—which is useful for web-uploading purposes (see the related section below)—but will also make your animation less smooth.
  • Click OK. You should now have imported each frame as its own layer. Finalize your animated GIF and Save for Web as usual.

Taking advantage of After Effects

Before opening it Photoshop, you can do preparatory editing in After Effects—e.g. trimming the clip, stabilizing shaky footage…—and export the edited version as an image sequence. Note that After Effects can import animated GIFs as footage, but it cannot directly export animated GIFs. You’ll need Photoshop for that.

Make your footage less shaky

  • Import your footage and make a new composition out of it.
  • With the footage selected in your timeline, go to After Effects’s top menu bar and select Animation > Warp Stabilizer VFX.
  • This should stabilize your video. Use the Warp Stabilizer effect panel to adjust the way your video stabilizes (e.g., You may want to change the method. It defaults to Subspace Warp but I often only want it to adjust for Position, Scale and Rotation so will change this setting).

Exporting video as Image Sequence

  • You can render the video as QuickTime format as usual, but saving as an image sequence can help streamline the process.
  • When done editing in After Effects, Add to Render Queue.
  • In the Render Queue panel, click Render Settings and choose Frame Rate > Use this frame rate to set it at 10. Hit OK.
  • Click Output Module. Under Format dropdown, change from QuickTime to PNG Sequence. Hit OK.
  • When you hit Render, the composition will be rendered as a series of PNG files—each frame as a separate image.
  • Open Photoshop. Go to File > Open, and select any of the rendered PNG files. Make sure to check the Image Sequence box.
  • When Frame Rate popup window appears, type in 10 as specified in After Effects render option.
  • Now you have the composition loaded in Photoshop. Do what you need to do edit, and Save for Web.

Keeping the file size small:

When viewing over the internet, smaller GIFs load quicker and play smoother than large ones, but making a file size smaller often means degrading the image quality of the GIF. That is, producing GIFs is always a bit of a compromise between good image quality and small file size.

Often, animated GIF are not meant to be a refined motion graphic piece, and certain weirdness and roughness can be a part of its charm. So don’t feel too much pressure to achieve super high quality of image or/and perfectly smooth motion (like what you’re looking to get when working in After Effects).

That said, it’s good to at least try for the highest quality possible, and there’s not a single consistent best practice for keeping file size small while keeping image quality high. Decide which website the GIF is being uploaded to—some websites have maximum or minimum file sizes—and try a few techniques.

  • General considerations for keeping file size down:
    • Number of frames: Fewer frames = smaller file size
    • Number of colors: Fewer colors = smaller file size
    • Image dimension: Smaller image size = smaller file size
    • Dithering: dithering is the process of using two colors to simulate the shade of a third color by placing the two colors close together. Given that GIFs support a limited number of colors, dithering is useful to achieve smoother image. But more dithering results in a larger file size. You can reduce file size by decreasing the percentage of dither. This can be changed in the Save for Web popup window, along with the number of colors and image size.

Uploading your GIFs to our Tumblr:

  • Go to the submit page on the Tumblr set up for this class. You can also access this page by clicking the + sign on the bottom left corner of the Tumblr’s homepage, and clicking Submit a GIF in that menu.
  • Upload your gif. Remember Tumblr’s max size for animated gifs: no larger than 3MB and no more than 540 pixels wide.
  • Write your name in the Caption field, so I know whose submission this is.
  • Fill out the rest of the required info (you’ll need to provide a name and an email address, or log into a personal Tumblr account) and hit Submit.
  • Your image won’t appear immediately since I have to approve all of the submissions to the Tumblr. Don’t worry, they’ll all be approved and up before the next class.

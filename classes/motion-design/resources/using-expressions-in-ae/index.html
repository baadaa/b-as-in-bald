Using Expressions in AE

Expressions are a way to animate layers in After Effects without manually placing and adjusting keyframes. Depending on what you're looking to do, expressions can be a fast and efficient way of working.

What is an Expression?

  • Expressions are based on JavaScript, a programming language, and can dynamically generate motion sequence based on relationships between layer properties.
  • It can be useful when creating a continuous motion through the length of your composition, or working on an animation that is very repetitive or/and based on numerical relationships.
  • Expressions can be applied to:
    • Any keyframe-able properties (e.g. position, scale, rotation…)
    • Any specific effect values applied to a layer
  • Very similar to blending modes in other Adobe programs

Using Expressions: Basics

  • Open Expression editing field by:
    • Hold down Alt/option key and click on the stopwatch icon, next to the specific property
    • OR click on the specific property in your timeline, and choose from top menu Animation > Add Expression
  • A code editing field will open in timeline area. If using an expression you’re already familiar with or/and an expression found online, you can just type it in here. When done typing, click outside of the code editing area.
  • While AE Expressions are based on JavaScript, you don’t have to write JS programs to use them.
  • After effects has a built in Expressions library that you can search through. Click the ‘arrow inside a circle icon’ located the furthest right in this expression’s timeline sublayer to access it.
    • This library shows you different behaviors/actions that you can apply, and the language that to use to make that happen.
    • To examine how applying expressions works, try a simple rotation expression in After Effects’ library.
    • It’s rarely helpful to just dig around Expression Library when you’re not looking for any specific type of action. If you have a specific type of complex or continuous movement in mind, search for an Expression that makes that specific motion happen.

Applying an Expression to a Property

  • Select the specific layer and property to apply the expression to, and select the icon that opens the expressions library.
  • For this example, choose Rotation property, and select Global > time in Expression Library.
  • Once you select time, text block will appear in this expression sublayer in timeline. Preview your composition to see how your layer is rotating.
  • This Expression simply applies the time value to Rotation property of the layer. That is, 1 degree rotation per every second. To adjust its speed, all you need is simple math. (Note that Expressions can involve a fair amount of simple math)
  • After Effects math symbols follow standard programming language convention as below:
    • +: addition
    • –: subtraction
    • *: multiplication
    • /: division
  • To increase the speed, use multiplication. Update the existing code as below: 
    time * 20
  • The updated code now applies time value multiplied by 20 as the rotation degree. You can adjust the multiplication number to change the speed now. You’ve applied properties of motions not with keyframes, but with code.
  • Now try applying Time expression to the other object’s rotation property, make it rotate at a different speed

Advanced Use of Expressions

  • Expressions can become much more complex than this simple example, and often need null objects to be incorporated.
    • A null object is a layer that contain only properties of motion and is invisible. They are typically used to control other layer(s), often using Expressions.
    • To do this,
      • Create a null layer, and add Expression Controls to it by Effects > Expression Controls > and select one.
      • Then, parent your individual layer to this null layer.
  • You can also link different layers together, so that one is referential of the other. For instance, using a percentage-based value, one layer’s property can be based on another layer’s property.
  • As with basically everything about After Effects, Expressions can get as complicated as you’d like them to be.

Tech Demo 2

Outlines and instructions for in-class tech demon

Layer Blending Modes

  • Very similar to blending modes in other Adobe programs
  • At the very bottom of your timeline panel, towards the left side, is a button that says Toggle Switches / Modes
  • When you click on this button, you have the option to change the mode of each layer by clicking the drop down next to Mode
  • You can change modes to adjust the way each layer blends or interacts with the layers beneath it (similar layer modes to other Adobe programs, like Multiply, Overlay, etc)
    • Note: there isn’t a direct way to add keyframes to blending modes—once you’ve picked one, it’s applied to that layer for the whole composition. If you want to work around this, you can create precomps of the layers involved in the blending effect (one percomp for each blending mode you wish to transition between) and use Transform > Opacity to transition between them
  • To get back to your switches (things like motion blur) just click the Toggle Switches / Modes button again

Layer Masks

  • Similar to masks in Photoshop or clipping masks in Illustrator—you draw a shape through which underlying footage/image/shape/animation) is visible, and the area outside of this shape is masked (hidden).
  • To implement layer masks:
    • Select the layer you wish to mask in your timeline, and draw a shape using the pen tool or a shape tool on top of it
      • Note: When applying a layer mask to a shape layer, an extra step is necessary. Before drawing a mask, make sure to select the Tool Creates Mask button—located at the very top of your screen in the same row as the pen tool, shape tool, etc, and looks like a black circle on a checkerboard pattern. Once clicked, it will turn blue.
    • If you have masked a layer that contains video or an animated composition/precomp—basically any moving footage—that layer will continue to play inside the mask
    • Note that under your masked layer in your Timeline, you now have a Mask 1 dropdown. You can use this to add keyframes to and adjust your layer mask over time.
    • Mask path can be used to adjust both the position and the shape of the path (you can animate the path using your pen / vertex tools just like you can do with a shape layer)
  • To move and adjust paths for the mask (this can be tricky):
    • If you just click and drag the shape with your selection tool, it will change the position of the entire layer (mask and background)
    • If you click once on your shape before attempting to move or change its path, you’ll then be able to select and adjust each individual vertex (anchor point)
    • If you select the specific mask sub-layer—for example, ‘Mask 1’—you will be able to change the position of your mask, but not of the image or footage that it is masking.
    • Adjusting the position of a layer mask with your selection tool can be a little tricky, so I tend to use my arrow keys for this. Set a mask path keyframe, scrub further along your timeline and use your arrow keys to move your layer mask to where you would like its new position to be. Note that the masked image does not move along with the mask.
  • Mask Feather can be used to adjust the sharpness of the mask’s edges
  • Mask Opacity can be used to adjust the opacity
  • Mask Expansion can be used to adjust the size of the mask
  • You can also add a limited number of blending modes to the mask, or inverse the masked area by checking the box next to Inverted

Track Mattes

  • You can also apply a track matte as a way of masking layers. The process is slightly different from making a simple shape layer mask, but it allows for greater flexibility and control.
  • To apply a track matte:
    • Add a layer of footage (or a composition, or precomp your existing layers) to your timeline, and add a shape layer on top normally.
      • Note: when applying track matte, it is important that the layer you plan to use as the mask be placed directly on top of the layer that it will be masking in your timeline window
    • Click Toggle Switches / Modes so that you can edit Modes
    • Select the layer to be masked (i.e. your layer of footage, NOT the shape that you want to become the mask)
    • Click this layer’s TrkMat dropdown menu, and choose to Alpha Mat the shape layer you want to use as the mask
    • Your shape layer should now be acting as a layer mask
      • Note: With a track matte mask, you can move around the masked layer around while keeping the mask still—which is not something you can do with a regular layer mask).
  • Another benefit of track mattes is that you can use letterforms as masks using this method. To do this, add a type later on top of whatever footage, etc, you want to mask and treat it in exactly the same way you would a shape layer you’re using as a track matte mask.

Parenting Layers

  • When you parent one layer to another, the child layer will follow all of the animated Transform properties of the parent layer.
    • Create a parent layer. The animated transform properties of this layer will be applied to all child layers. You can make this layer invisible, if you don’t want this parent layer to be seen.
    • Click Toggle switches / modes so that you’re viewing modes.
    • Select the Parent dropdown menu in your child layer(s) and change it from None to the name of the parent layer
    • You can still animate the child layer independently in addition to animating the parent layer. Animating the child layer will have no effect on the movement of the parent layer.

Type Presets

  • Animation presets are predetermined specific configurations of layer properties, behaviors, and animations that After Effects has set for you. You can apply animation presets to the layers in your timeline. You may want to experiment with type-specific presets in your kinetic type piece:
    • Open the Effects & Presets panel
    • Click the arrow next to Animation Presets to open that dropdown
    • Click the arrow next to Text to open the text-specific animation presets
    • Play around with the many options available to you. To apply a preset, select your layer of type in the timeline and then double click on the specific preset you wish to apply
    • Once you have applied a preset to a layer, that layer will have an additional dropdown called Animator in the timeline. This is where you can set keyframes to make minor adjustments to your preset (the ways in which you are able to adjust the animation varies by preset).
    • Note: do not rely ENTIRELY on type presets when working on a kinetic type piece. While they can be useful and save you a lot of time if you’re looking to create the specific kind of motion that the preset creates, you generally have a limited control over the preset. I tend to use presets sparingly.

Converting Shape Layers to Editable Shape paths

  • You can convert your shape layers—_drawn with the rectangle tool, circle tool, polygon tool, etc.—to a shape with an _editable path.
  • Draw your shape using the appropriate shape tool, as usual
  • Open the Contents dropdown of the layer in timeline, and then open the dropdown for the specific shape you wish to convert.
  • Control + click on that shape’s path. For example, if a single circle is drawn this shape layer, choose Ellipse Path 1.
  • Select Convert to Bezier Path. Your shape now has an editable path.
  • Note: In After Effects terms, this is called converting a parametric shape layer path into a Bezier path.

Morphing between Shapes

  • This is a technique you can use to morph between different shapes that’s a bit quicker and more efficient than moving your vertex points around by hand.
    • Draw two shapes you want to morph between (e.g. a square, a circle, an amorphous blob) each in their own shape layer.
    • If drawn using a shape tool (circles, rectangles, etc), convert their paths to bezier paths using the method above.
    • Set a Path keyframe for each shape
    • Select this keyframe on one layer and copy it. Then scrub forward in time, select your other shape layer’s path sublayer, and paste. This shape layer should now morph between its original shape
    • Repeat with as many shapes for as many transformations you’d like. Try changing transform properties (e.g. rotation, scale, etc) in addition to this shape morph for more dramatic effects.

Line End Points

  • Use this to keyframe the ends of a line you’ve drawn, to animate them appearing/disappearing.
    • Make a shape layer, and draw a line using your pen tool.
    • Click on Add: (>) which appears to the right of that shape layer’s Content dropdown in the timeline
    • Select Trim Paths. You should now have an additional timeline dropdown menu for this layer called Trim Paths 1.
    • Open Trim Paths layer. Here you can adjust the start and end point of your line and can offset the line from the drawn path.
    • Can add keyframes to animate the line, make it draw on, slide off (using start/end settings)

Write on Text

  • This creates the illusion that your text is being written/drawn on the screen.
    • Set your type as you want it to appear
    • With your type layer selected, click Pen tool to draw over your letterforms as if you are writing them by hand.
      • Make sure you do this in the same layer as your type, so that it acts as a layer mask and not as its own shape layer)
    • Once you have traced all of your type, with this layer selected, click Effect > Generate > Stroke on top bar menu.
    • In the Stroke panel that appears, change the brush size so that it’s large enough to entirely cover up the letterforms beneath
    • Change the Paint Style setting from Original Image to Reveal Original Image
    • You can now use the Start and End options (in the Stroke panel). Set keyframes like when applying trim paths to animate the type so that it looks like it’s being drawn on
    • Write on text can be a little tricky. A useful tutorial in case you’re still having trouble can be found here
